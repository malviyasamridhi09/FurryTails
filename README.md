🐾 Paws & Claws – Pet Care Website
A modern and interactive Pet Care Portfolio Website built using HTML, CSS, and JavaScript.
This project showcases a visually appealing design with smooth animations, responsive layout, and engaging UI components focused on pet adoption and pet care services.
This is my first front-end portfolio project, created to demonstrate creativity, design skills, and basic web development concepts.
(Project source: )
🌟 Features
✨ Attractive animated UI with gradient themes
✨ Fully responsive design for mobile, tablet, and desktop
✨ Interactive pet cards with adoption demo alerts
✨ Smooth scrolling navigation
✨ Animated service and testimonial sections
✨ Contact form with simulated submission feedback
✨ Floating pet icons and background animations
✨ Clean and modern layout using Flexbox & Grid
🛠️ Technologies Used
HTML5
CSS3 (Animations, Gradients, Flexbox, Grid)
JavaScript (DOM manipulation & interactions)
Google Fonts
Font Awesome Icons
